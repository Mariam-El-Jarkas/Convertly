## ğŸ§­ Usage

1. **Open the app.**  
2. **Read the short intro** about Convertly.  
3. **Select a conversion category** (Time, Length, Weight, Temperature).  
4. **Enter a value** in the input field.  
5. Select **From** and **To** units from the dropdown menus.  
6. Tap **Convert** to see the result.  
7. Tap the **Reset icon** in the AppBar to clear the input and perform a new conversion.
